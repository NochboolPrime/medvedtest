# Документация по цветовым схемам сайта

## Описание системы тем

Сайт поддерживает две темы:
- **Темная тема** (по умолчанию): темно-синий фон #1a1f2e
- **Светлая тема**: белый/светло-серый фон

Переключение тем происходит с помощью кнопки в header (иконка солнца/луны).

---

## Основные цвета

### Темная тема (по умолчанию)
- Основной фон: `#1a1f2e` (темно-синий)
- Карточки: `#1a1f2e` (темно-синий, тот же что и фон)
- Текст: `#ffffff` (белый)
- Второстепенный текст: `#B7C5DB` (светло-голубой)
- Акцентный цвет: `#B19D76` (бежевый/золотой)
- Границы: `#B7C5DB` с прозрачностью

### Светлая тема
- Основной фон: `#ffffff` (белый)
- Карточки: `#f8f9fb` (очень светло-серый)
- Текст: `#1a1f2e` (темно-синий)
- Второстепенный текст: `#6b7280` (серый)
- Акцентный цвет: `#B19D76` (бежевый/золотой - тот же)
- Границы: `#e5e7eb` (светло-серый)

## Компоненты и их цвета

### 1. Header (components/header.tsx)
**Темная тема:**
- Фон: `bg-[#1C2433]/98` с blur эффектом
- Текст логотипа: белый
- Навигация: `text-[#B7C5DB]`, при hover - белый
- Телефон/Email: белый и светло-голубой

**Светлая тема:**
- Меняется через классы с модификатором без `dark:`
- Примеры см. в коде компонента

### 2. Hero (components/hero.tsx)
**Темная тема:**
- Фон секции: `dark:bg-[#1a1f2e]`
- Градиент поверх изображения: `dark:from-[#1a1f2e]/95 dark:via-[#1a1f2e]/70`
- Заголовок: белый через токен `text-foreground`
- Описание: светло-голубой через `text-muted-foreground`

**Светлая тема:**
- Фон: `bg-card` (белый)
- Градиент: `from-card/95 via-card/70`
- Текст: темный автоматически

### 3. About (components/about.tsx)
**Темная тема:**
- Фон секции: `dark:bg-[#1a1f2e]`
- Карточки стратегии: `dark:bg-[#1a1f2e]`
- Карточки сертификатов: `dark:bg-[#1a1f2e]`
- Номера принципов: `text-[#B19D76]/30`

**Светлая тема:**
- Фон: `bg-muted` (светло-серый)
- Карточки: `bg-card` (белый)
- Текст: темный автоматически

### 4. Services (components/services.tsx)
**Все темы:**
- Использует семантические токены (bg-background, text-foreground)
- Автоматически адаптируется
- Номера услуг: `text-muted-foreground/40`

### 5. Contact (components/contact.tsx)
**Темная тема:**
- Фон секции: `dark:bg-[#1a1f2e]`
- Градиент: `dark:from-[#1a1f2e] dark:via-[#1a1f2e] dark:to-[#1a1f2e]`
- Форма: `dark:bg-[#1a1f2e]/80`

**Светлая тема:**
- Фон: `bg-muted` (светло-серый)
- Форма: `bg-card/80` (белый с прозрачностью)

### 6. Footer (components/footer.tsx)
**Темная тема:**
- Фон: `dark:bg-[#1a1f2e]`
- Текст: белый через токены
- Ссылки: светло-голубой при hover

**Светлая тема:**
- Фон: `bg-card` (белый)
- Текст: темный автоматически

## Как менять цвета

### Изменить основной цвет темной темы (#1a1f2e):
1. Найдите в компонентах все `dark:bg-[#1a1f2e]`
2. Замените на нужный цвет, например `dark:bg-[#новый_цвет]`

### Изменить акцентный цвет (#B19D76):
1. Найдите все `bg-[#B19D76]` или `text-[#B19D76]`
2. Замените на нужный цвет

### Изменить цвета карточек:
- Темная тема: замените `dark:bg-[#1a1f2e]` в карточках
- Светлая тема: замените `bg-card` на `bg-[#нужный_цвет]`

## Семантические токены (из globals.css)

Используются для автоматической адаптации:
- `bg-background` - основной фон
- `bg-card` - фон карточек
- `bg-muted` - приглушенный фон
- `text-foreground` - основной текст
- `text-muted-foreground` - второстепенный текст
- `border-border` - границы
- `bg-primary` - акцентный цвет кнопок

Эти токены автоматически меняются при переключении темы в файле `app/globals.css`.
